Libre Docs Advanced Freedom

Libre Docs does not store your data. You need to bring your own remote storage to the party. There are several ways for you to get a remote storage account that will work in combination with Libre Docs:
  • Get your own CouchDB instance at Iris Couch. Even though this is only one of the options, we recommend Iris Couch because they are nice people, who care about the open web and about decentralization of control. You can always start with this option, and switch to a more advanced one later. To sign up, go back and click the 'Easy Freedom' button.
  • Run CouchDB yourself and run our Couch Pimper against it.
  • Run ownCloud and pay special attention to point 4, as well as to the last paragraph, of the instructions.
  • You can also link storage to your WordPress installation, using this (experimental) WordPress plugin. Note that you will still need either CouchDB or ownCloud somewhere to work in combination with this.
  • Develop your own software, or make existing software compatible with the remoteStorage standard.
